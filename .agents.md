# Codex Pro Max HITL Alias

This project uses `AGENTS.md` as the active Codex instruction file.

The Codex Pro Max UI now uses one primary `Send to Codex` button. Treat normal UI input as `INSTRUCTION_RECEIVED`: read `instruction.txt`, decide what the user wants, write the answer or result back to the selected run's `output.md` and `progress.md`, then wait again.

Do not answer in the chat window or stop the workflow unless the user explicitly says to stop, finish, end, or disable HITL.
